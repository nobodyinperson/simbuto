% simbuto(1) | simple budgeting tool

NAME
====


**simbuto** - a simple graphical budgeting tool

SYNOPSIS
========


**simbuto** [file]

FILES
=====


|   File     | Purpose                                                           |
|------------|-------------------------------------------------------------------|
|`~/.simbuto`| per-user dotfolder - created automatically on application startup |


ENVIRONMENT
===========


| Variable                | Impact                           |
|-------------------------|----------------------------------|
|**LANG** or **LANGUAGE** | UI language, defaults to English |
